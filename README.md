# NAMS

**Next Asset Management System** — 수입·지출·투자를 한곳에서 관리하는 개인 자산 관리 웹앱.

GitHub Pages 같은 정적 호스팅에서 동작하며 모든 데이터는 사용하는 기기에 자동 저장됩니다. 별도의 서버나 로그인이 필요 없습니다.

---

## 주요 기능

### 💰 수입 / 지출
- 날짜·카테고리·금액·메모로 거래를 입력, 수정, 삭제
- 카테고리 직접 추가·이름변경·삭제 (입력 시 자동완성 지원)
- 지출 거래에는 결제수단 지정
- 달력 형태의 직관적인 날짜 선택
- 카테고리 이름을 바꾸면 기존 거래·고정지출에 일괄 반영

### 📌 고정지출
- 월세·구독료·보험료처럼 매달 반복되는 지출을 위젯으로 등록 (이름·카테고리·색상·결제일·금액·결제수단)
- 결제일 D-day 표시, 카테고리별 구성 비율 막대
- **위젯 묶기** — 같은 카테고리의 위젯을 드래그해 겹치면 하나의 묶음으로 합쳐져 금액 합계를 보여줍니다. 묶음에 마우스를 올리면 내부 항목이 펼쳐지고, × 버튼으로 묶음을 해제할 수 있습니다
- 달이 바뀌면 그 달의 고정지출이 기록으로 자동 보존

### 📈 투자
- **예금 · 적금 · 주식** 세 종류의 상품 관리
- 예금·적금: 원금·이율·기간으로 단리/복리 만기 예상액을 계산하고 진행률 표시
- 주식: 종목 코드로 **실시간 시세 조회**(1분 주기 자동 갱신), 평가손익·수익률 추적
- 투자 원금 합계 · 현재 평가액 · 평가손익 한눈에 요약

### 🧪 투자 시뮬레이션
- 초기 투자금·기간·월 적립금·기대수익률·변동성·현금 비중 입력
- 일시 투자, 적립식 투자, 목표 비중 리밸런싱, 이동평균선, 손절/익절 전략 비교
- 거래 수수료·세금·슬리피지를 반영한 결과 계산
- 최종 자산, 총수익률, 연환산 수익률, 최대 낙폭, 변동성, 샤프 비율 표시
- 자산 변화, 수익률/낙폭, 투자자산/현금 비중, 거래 타이밍 차트와 목록
- 시나리오 저장·불러오기·복제·삭제 및 벤치마크/저장 시나리오 비교

### 🧮 지출 관리
- 결제수단별 사용액 분석, 한도·목표 대비 사용률 표시
- 월 단위 조회, 결제수단 추가·수정·통합
- 결제수단 변경 시 모든 거래에 일괄 반영

### 📊 그래프 요약
- 총수입 · 총지출 · 투자 평가액 · 순자산 요약 카드
- **미래 자산 추이** 예측 그래프 (1·3·5·10년 — 현금/예금/적금/주식 누적)
- 월별 수입·지출 추이, 수입 대비 지출, 카테고리별 비중(파이 차트), 누적 현금 흐름
- 연도별 필터

## 데이터 저장 & 백업
- 모든 데이터는 정적 배포에 맞춘 브라우저 문서 저장소를 통해 localStorage에 자동 저장됩니다
- **JSON 내보내기 / 가져오기** — 현재 브라우저 문서를 백업 파일로 저장하고 그대로 복원
- **구글 드라이브 백업** — 구글 계정으로 로그인해 데이터를 드라이브에 저장·복원 (선택 기능, `VITE_GOOGLE_CLIENT_ID` 설정 필요)

## 화면 & 사용성
- **다크 / 라이트 모드** 전환 (Discord 스타일 다크 테마)
- 탭(스테이지) **표시 여부 토글 및 순서 변경** — 탭 영역을 우클릭해 설정
- 모바일까지 대응하는 반응형 레이아웃

---

## 기술 스택
- **React 18** + **Vite 6**
- **Recharts** — 차트 렌더링
- 상태·저장: 커스텀 훅 + 브라우저 문서 저장소(localStorage, 백엔드 없음)

## 시작하기
```bash
npm install      # 의존성 설치
npm run dev      # 개발 서버 실행
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```
